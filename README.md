## üéØ Overview

GreyWaken Medical teaches healthcare professionals how to use AI safely and effectively in clinical practice. The course covers everything from AI fundamentals to advanced prompt engineering, with a strong emphasis on patient privacy and HIPAA compliance.

**Duration:** 6 hours (self-paced)  
**Certification:** IIHMRB Certificate of Completion  
**Target Audience:** Healthcare professionals

## ‚ú® Key Features

- **7 Comprehensive Modules** - From basics to advanced implementation
- **Medical Analogies** - AI concepts explained through familiar healthcare terms
- **HIPAA-First Approach** - Patient privacy emphasized throughout
- **Hands-On Learning** - 5 practical exercises + comprehensive capstone
- **AI Tutor** - 24/7 assistance with context-aware responses
- **Adaptive Assessments** - Study/Test modes with progressive hints
- **Progress Tracking** - Visual journey map and completion metrics
- **Admin Dashboard** - User management, analytics, curriculum control

## üìö Curriculum

### Module 1: AI in Your Practice Today (18 min)
Introduction to AI applications in healthcare, safety rules, practical examples

### Module 2: How AI Thinks (26 min)
Pattern recognition, models, parameters, tokens, context windows

### Module 3: Protecting Patient Privacy (20 min)
HIPAA compliance, 18 identifiers, de-identification, BAAs

### Module 4: Advanced Prompt Engineering (22 min)
Documentation workflows, clinical decision support, patient communication

### Module 5: AI Safety & Bias (19 min)
Hallucinations, bias recognition, ethics framework

### Module 6: Building Workflows (24 min)
Workflow analysis, tool selection, ROI measurement

### Module 7: Capstone Project (90 min)
Comprehensive assessment, practical exercises, action plan creation

**Total:** 219 minutes | 32 lessons | 7 quizzes | 5 exercises

## üöÄ Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- AI API key (Gemini or Groq)

### Installation

```bash
# Clone repository
git clone https://github.com/satishskid/neurospark.git
cd neurospark

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173`

1. Click Settings (‚öôÔ∏è) in header
2. Choose AI provider (Gemini or Groq)
