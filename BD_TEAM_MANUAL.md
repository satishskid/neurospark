## üöÄ User Onboarding Process

### Individual Users

**Step 1: Registration**
1. User visits: `https://greywaken.greybrain.in`
2. Clicks "Begin Learning"
3. Selects curriculum (Medical AI or Basics of AI)
4. Signs in with Google or Email

**Step 2: Whitelist Verification**
- User email must be whitelisted
- If not whitelisted ‚Üí "Contact support" message
- Whitelisted users ‚Üí Proceed to course

**Step 3: Learning Journey**
- Complete onboarding tutorial
- Progress through modules
- Track completion in dashboard
- Earn certificate upon completion

### Institutional Users

**Step 1: Partnership Agreement**
1. BD team negotiates terms
2. Contract signed
3. User list provided

**Step 2: Bulk Onboarding**
1. BD team sends user email list to tech team
2. Tech team adds to whitelist
3. Users receive welcome email (manual for now)
4. Users can register and start learning

**Step 3: Monitoring**
1. Admin tracks user progress
2. Completion reports available
3. Issue tracking for support
